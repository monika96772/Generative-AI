**Procurement and Inventory Analysis using Generative AI (ChatGPT)**

Overview
This Proof of Concept demonstrates how injecting SAP S/4HANA Materials Management (MM) context into ChatGPT prompts transforms generic AI responses into actionable ERP strategies (e.g., triggering PRs, evaluating ATP, adjusting MRP parameters).

Tech Stack
* AI: Generative AI (ChatGPT), Prompt Engineering
* Domain: SAP S/4HANA MM, MRP Planning, Inventory Optimization

Methodology
1. Baseline Prompt: Generic procurement prompt yields basic math and standard advice.
2. Context-Injected Prompt: SAP-specific context generates targeted actions (e.g., ME61 vendor evaluation, EX/FX lot-sizing).

Repository Files
* Procurement_and_Inventory_Analysis_GenAI.pdf` - Visual case study & prompt evolution.
* poc_dataset.csv - Sample micro-dataset.
